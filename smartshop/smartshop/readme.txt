Για να λειτουργήσει η εφαρμογή θα πρέπει:

1. Να γίνει import το αρχείο e_shop.sql από το φάκελο /database στη βάση δεδομένων σας.
2. Να γίνει επεξεργασία του αρχείου /includes/db_connect.php, όπου πρέπει να δηλωθούν σωστά οι εξής σταθερές:΄
DB_SERVER: διεύθυνση του MySQL server σας
DB_DATABASE: το όνομα της βάσης που θα χρησιμοποιήσετε
DB_USERNAME: το όνομα που έχετε δηλώσει ως χρήστη για σύνδεση στο server
DB_PASSWORD: το password του παραπάνω χρήστη 

Έχει γραφτεί και λειτουργεί με PHP Έκδοση 7.4.10
